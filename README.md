Project Title: MLOps Structured Project



Objective:

The aim of this experiment is to organize a machine learning project into a proper folder structure and train a basic model.



Dataset:

The Iris dataset is used, loaded from an online source.



Steps to Run:

1\. Install required libraries using requirements.txt

2\. Run the training script from source folder

3\. The trained model will be saved in the saved\_models folder
